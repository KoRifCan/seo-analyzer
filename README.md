# SEO Analyzer 🔍

Complete website SEO analysis — meta tags, headings, images, links, content, technical, schema, security, accessibility, mobile, social, and performance. All in one HTML file. No server needed.

**🔗 Live:** https://korifcan.github.io/seo-analyzer/

## Features

| Category | Checks |
|----------|--------|
| **Meta** | Title (length), meta description (length, quality), keyword match, OG tags (title/desc/image/url), Twitter card, favicon |
| **Headings** | H1-H6 structure, missing H1, multiple H1, heading hierarchy skip |
| **Images** | Total, with alt, missing alt, empty alt, broken src |
| **Links** | Total, internal/external, nofollow, anchor text quality |
| **Content** | Word count, reading time, sentences, paragraphs, content ratio, keyword density (top 5), keyword stuffing detection, placeholder content |
| **Technical** | SSL, viewport, charset, language, canonical, robots meta, robots.txt, sitemap.xml |
| **Schema** | JSON-LD detection, schema types extraction, validation |
| **Security** | HTTPS, CSP, X-Frame-Options, form input check |
| **Social** | OG completeness, Twitter card, social description quality |
| **Performance** | HTML size, estimated requests, render-blocking scripts, lazy loading, inline styles |
| **Accessibility** | Alt text, heading structure, language, viewport, form labels |
| **Mobile** | Viewport, DOM size, touch targets |
| **URL** | Cleanliness, parameters, underscores, uppercase, www |

## Tech

- Pure HTML/CSS/JS (single file)
- CORS proxies with auto-fallback
- GitHub Pages hosting
- Dark/Light theme

## Cara Pakai

1. Open https://korifcan.github.io/seo-analyzer/
2. Enter a URL
3. Click **Analyze**
4. Browse results by category tab
